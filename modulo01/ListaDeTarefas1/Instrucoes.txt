Instruções para a execução dos exercícios:

Localize a pasta "ListaDeTarefas1" e selecione a opção "Abrir com Code"
Abra um novo terminal no VsCode e execute o comando "npm install prompt-sync"
Selecione o exercício que desejar e rode o código (é possível rodar o código com o atalho CTRL + ALT + N)
Preste atenção para estar no "Terminal" e não no "Output"
Do exercício 1 ao 13, caso seja um número decimal digite com "." ao invés de "," 

Exercício 1: Insira a temperatura que deseja converter
Exercício 2: Insira os valores desejados que a conversao em % será feita
Exercício 3: Basta inserir os valores
Exercício 4 e 5: Insira a primeira e a segunda nota
Exercício 6: Insira as medidas dos lados do triângulo
Exercício 7: Insira o total de maçãs desejadas
Exercício 8: Insira o primeiro e o segundo valor para ordenar de forma crescente
Exercício 9: Insira o código de origem (valor) que mostrará a região correspondente
Exercício 10: Insira um número para ele repetir 10 vezes
Exercício 11: Insira quantos valores quiser para descobrir se é par ou ímpar, digite "sair" para finalizar o algoritmo
Exercício 12: Basta rodar o código
Exercício 13: Insira um valor, ele será o multiplicador dos números 1, 2, 3, 4 e 5
Exercício 14: Insira os números decimais desejados, mostrará a média aritmética deles. Digite 0 para finalizar o algoritmo
Exercício 15: Semelhante ao exercício 14, insira os números decimais e os pesos respectivamente e ele mostrará a média ponderada. Digite 0 para finalizar o algor
Exercício 16: Basta rodar o código
